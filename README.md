# Admin Dashboard Template

This is a **customized admin dashboard** built with **React 19** and **Tailwind CSS v4**, designed to manage data, users, and analytics efficiently. The dashboard is responsive, modern, and includes essential UI components for any admin panel or web application.

## Key Features

- Fully responsive sidebar and navigation
- Tables, charts (Line & Bar) for data visualization
- Profile management and authentication forms
- Alerts, dropdowns, modals, buttons
- FAQ, accordion, testimonials, and carousels
- Dark Mode support
- Easy to extend and customize
